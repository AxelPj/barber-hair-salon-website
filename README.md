# Les Maîtres de L'HAIR — Barber & Hair Salon

Site vitrine moderne et immersif que j'ai développé pour un salon de coiffure et barbier. L'objectif était de concevoir une interface fluide, épurée et entièrement responsive adaptée au secteur du soin et de la beauté.

[Voir le site en ligne](https://votre-lien-vercel-ou-netlify.com)

---

## Fonctionnalités du projet

*   **Navigation par onglets (SPA) :** J'ai mis en place une structure de Single Page Application pour permettre de basculer instantanément d'une section à l'autre sans aucun rechargement de page via React.
*   **Transitions d'arrière-plan fluides :** Gestion dynamique du header qui passe d'une vidéo d'ambiance sur l'accueil à une image fixe sur les autres pages avec un effet de fondu pour adoucir le changement visuel.
*   **Animations au défilement :** Utilisation de l'API native Intersection Observer pour déclencher l'apparition des éléments (images, blocs de texte) par les côtés dès qu'ils entrent dans l'écran de l'utilisateur.
*   **Fiche de contact et horaires :** Intégration d'une section complète regroupant les horaires d'ouverture, les coordonnées de l'établissement et un formulaire de contact stylisé.
*   **Design sombre et épuré :** Interface construite en Mobile-First avec Tailwind CSS, basée sur une charte graphique sombre (zinc-950) rehaussée par des touches ambrées pour un rendu premium.

## Technologies que j'ai utilisées

*   **Framework :** Next.js (App Router)
*   **Bibliothèque :** React
*   **Styles :** Tailwind CSS
*   **Icônes :** SVG natifs / Lucide React

## ✨ Fonctionnalités

*   **Single Page Application (SPA) :** Navigation instantanée entre les onglets grâce à la gestion d'états React (`useState`), sans rechargement de page.
*   **Arrière-plan dynamique :** Transition fluide avec un effet de fondu (`fade-in`) entre un fond vidéo immersif pour l'accueil et une image premium pour les autres sections.
*   **Animations au défilement (Scroll Animations) :** Intégration de l'API native `IntersectionObserver` pour faire apparaître les éléments textuels et visuels par les côtés de manière ultra-fluide au fil du scroll.
*   **Fiche de contact complète :** Section intégrée regroupant les horaires d'ouverture, les coordonnées du salon et un formulaire de contact stylisé.
*   **Design Responsive & Mode Sombre :** Interface pensée en *Mobile-First*, optimisée pour smartphones, tablettes et ordinateurs, basée sur une palette de couleurs sombres et ambrées (`zinc-950`, `amber-400`).
